Source Code for Data Structures and Algorithm Analysis in C++ (Fourth Edition)

Here is the source code for Data Structures and Algorithm Analysis in C++ (Fourth Edition), by Mark Allen Weiss, which will be published in July 2013. The materials here are copyrighted.

Many C++11 features are used. I have successfully compiled and tested the programs under g++ 4.6.2.
